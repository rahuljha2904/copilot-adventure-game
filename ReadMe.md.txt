# Choose Your Own Adventure Game (Python)

This is a simple command-line based adventure game built using Python.

## About the Project
The game presents the player with multiple choices, and each decision leads to a different outcome.

## GitHub Copilot Usage
GitHub Copilot was used to generate:
- Function structure
- Conditional branching logic
- User input handling

By writing comments and prompts, Copilot suggested code that was refined to complete the game.

## How to Run
```bash
python adventure.py
